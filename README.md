# Contributor Guide: Building the HurricaneMetre 'Dumb Shapes'

Welcome to HurricaneMetre! We're stoked you're here to help build the global map.

## Getting Started
* Fork the [HurricaneMetre repo](https://github.com/mitchgibbonsemails-stack/HurricaneMetre)
* Clone your fork: `git clone https:                                                
* Install dependencies: `npm install` (or `yarn install`)

               
* Create a new branch: `git checkout -b feature/my-new-shape`
* Make changes and commit: `git commit -m "Add my new shape"`
* Push to your fork: `git push origin feature/my-new-shape`
* Open a pull request

## Guidelines
* Follow the BSD 3-Clause License terms
* Keep code clean and readable
* Test your changes thoroughly

## What to Work On
* 'Dumb Shapes' for the global map
* Improvements to the 3D map (The Real Twin)
